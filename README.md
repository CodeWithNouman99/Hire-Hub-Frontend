# HireHub 💼

A modern job portal web application built with React and Tailwind CSS that helps users discover job opportunities, explore job details, and interact with an AI-powered assistant for career-related guidance.

> ⚠️ This project is primarily frontend-focused and uses external APIs for job data and authentication services.

## ✨ Features

* 💼 Browse available job listings
* 📄 View detailed job information
* 🔍 Search jobs efficiently
* 🎯 Filter jobs based on preferences
* 📝 Apply for jobs
* 🔐 Secure authentication with Clerk
* 🤖 AI-powered chatbot for predefined career and job-related assistance
* 📱 Fully responsive design for mobile, tablet, and desktop devices
* ⚡ Fast navigation using React Router

---

## 🛠️ Tech Stack

| Technology        | Purpose                            |
| ----------------- | ---------------------------------- |
| React.js          | Component-based UI development     |
| Tailwind CSS      | Styling and responsive design      |
| React Router      | Client-side routing                |
| Clerk             | Authentication and user management |
| JavaScript (ES6+) | Application logic                  |
| Vite              | Development and build tool         |
| RapidAPI          | Job data source                    |

## 🌍 APIs & Services

### RapidAPI

Used to fetch job-related data dynamically.

### Clerk Authentication

Provides secure user authentication and account management.

### AI Chatbot

Integrated chatbot that answers predefined career and job-related questions.

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

```bash
# Clone the repository
git clone https://github.com/CodeWithNouman99/Hire-Hub-Frontend

# Navigate to the project folder
cd hirehub

# Install dependencies
npm install

# Start development server
npm run dev
```

Open:

```text
http://localhost:5173
```

in your browser.
